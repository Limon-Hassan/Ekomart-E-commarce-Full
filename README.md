# 🍔 Evato E-Commerce (Food Selling Platform)

## 🚀 Live Demo
https://ekomart-e-commarce.withlimon.workers.dev/

---

## 📌 Description
Evato is a full-stack food e-commerce web application where users can browse products, add items to cart, and complete orders with secure online payment. It includes a powerful admin dashboard to manage products, categories, and orders efficiently.

---

## ✨ Features

### 👤 User Side
- Google Login Authentication
- Secure Authentication & Authorization
- Add to Cart & Checkout System
- Online Payment Integration (Stripe)
- Responsive & User-Friendly UI

### 🛠 Admin Dashboard
- Manage Products (Add, Update, Delete)
- Manage Categories
- Order Management System
- View Detailed Order Information

### ⚡ Advanced Features
- Real-time updates using Socket.io
- Global state management with Redux
- Clean and scalable architecture

---

## 🛠 Tech Stack
